Project name: "Taxi Station".

Enteties are main classes with variables that we will store  and retrieve from database. There are 4 of them : Car, Driver, Order, User.

Interface Repository are interface that we used in order to overide our construct. 

Repository is class that has method with commands like insert , select etc.

IDB is interface that helps with connection to Database.

PostgreDB is a class that implements and retrieves from login , password.

Controller is a class that notifies us if we created new object in our database.

Application retrives variables that we inserted with usage of our console.

Main uses all classes as one mechanism.
